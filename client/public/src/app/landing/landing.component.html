Polygon Battle

I have been working on this html browser game in my free time (who has that?) and I am having an absolute blast with it.

Currently I have implemented the following features:

  • Collision Detection
    • Player object to walls.
    • Player object to spheres.
    • Spheres to walls.
    • Spheres to spheres.
    • Projectiles to spheres.
  • Player Movement.
  • Player Shooting.
  • Pause.
  • Levels / Rounds.
  • End game.
  • Replay.
  • Player enhancment:
    • Change number of sides
    • Change projectile speed
    • Change cooldown of shooting

I look forward to finding more time to continue tweaking this game as a platform to experminet with canvas and what's possible in browser based games.

A few items on my immediate wish list to add as soon as possible.

  • Add mass to my collision handling for sphere to sphere interaction.
  • Add gravity to the player object so spheres are attracted to it with increasing effect as sphere size decreases.
  • Upon destruction create smaller spheres from large spheres.
  • Option to toggle sandbox mode vs. story mode.
    • Story Mode begins with triangle only able to shoot from one side.
    • Ability to spend points to updrade number of sides, and number of sides that attack.
    • Spend points / currency to upgrade attack speed and refresh rate.
